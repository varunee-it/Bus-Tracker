# 🚌 Bus Tracker

A modern Android-based bus tracking application designed to simplify student transportation with live bus location tracking, route management, and realtime updates.

---

## ✨ Features

- 📍 Real-time bus location tracking
- 🗺️ Google Maps integration
- 🔐 Firebase-powered authentication and backend support
- 🚌 Route and bus management
- 📲 Clean and user-friendly interface
- ⚡ Fast and responsive Android experience

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| Java / Kotlin | Android App Development |
| Firebase | Authentication & Realtime Database |
| Google Maps API | Live map visualization |
| Android Studio | Development Environment |
| Gradle | Build Automation |

---

## 📂 Project Structure

```bash
Bus-Tracker/
│
├── app/                          # Main Android application source code
│   ├── src/main/java/            # Java/Kotlin backend logic and activities
│   ├── src/main/res/             # XML layouts, drawables, fonts, UI resources
│   ├── AndroidManifest.xml       # Application configuration and permissions
│   └── google-services.json      # Firebase configuration file
│
├── gradle/                       # Gradle wrapper configuration
├── build.gradle                  # Root-level build configuration
├── settings.gradle               # Project module settings
├── gradlew                       # Gradle wrapper executable (Linux/Mac)
├── gradlew.bat                   # Gradle wrapper executable (Windows)
├── .gitignore                    # Files and folders ignored by Git
└── README.md                     # Project documentation
```

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/varunee-it/Bus-Tracker.git
```

### 2. Open in Android Studio

- Open Android Studio
- Click on “Open Existing Project”
- Select the cloned project folder

### 3. Configure Firebase

- Create a Firebase project
- Add Android app configuration
- Place `google-services.json` inside:

```bash
app/
```

### 4. Sync Gradle

Allow Android Studio to sync all dependencies.

### 5. Run the Application

Connect an emulator or Android device and run the app.
